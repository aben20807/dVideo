## Recording  (開發記錄)  
利用notepad++編譯程式，即可一支.exe行走天下  
使用時可以直接下載 dVideo.exe 點擊使用  
或下載含 cygwin 相關 .dll 的 download.zip 解壓縮後使用  

## Some Tip  (技巧)  
需要會使用瀏覽器的進階功能--"檢查".  

## file use  (檔案)   
複製網頁的程式碼貼到文字文件，並改名成"ss.in" (不含").  
執行程式後會得到一個叫"ss.out"的檔案，然後網址會在裡面  

## p.s.  (附註)   
Windows的換行符號是\r\n，如果要在Linux下執行可能需要改成\n
